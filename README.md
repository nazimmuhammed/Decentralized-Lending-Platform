# 🚀 Decentralized Lending Platform

A blockchain-powered **DeFi (Decentralized Finance)** application that enables users to securely lend and borrow cryptocurrency assets without traditional banks or intermediaries. The platform utilizes smart contracts to automate lending, borrowing, collateral management, and repayments while ensuring transparency and security.

---

## ✨ Features

🔹 Secure Cryptocurrency Lending & Borrowing
🔹 Smart Contract-Based Loan Processing
🔹 Collateral-Backed Loans
🔹 Automated Interest Calculation
🔹 MetaMask Wallet Integration
🔹 Transparent Blockchain Transactions
🔹 Decentralized & Trustless Architecture
🔹 User-Friendly Web Interface

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/nazimmuhammed/Decentralized-Lending-Platform.git
cd Decentralized-Lending-Platform
```

### 2️⃣ Install Dependencies

```bash
npm install
```

---

## 🔨 Compile Smart Contracts

```bash
npx hardhat compile
```

---

## 🧪 Run Tests

```bash
npx hardhat test
```

---

## 🏗️ Start Local Blockchain

```bash
npx hardhat node
```

---

## 🚀 Deploy Smart Contracts

```bash
npx hardhat run scripts/deploy.js --network localhost
```

---

## 💻 Run Frontend

```bash
cd frontend
npm install
npm start
```

Open in your browser:

```text
http://localhost:3000
```

## 🔄 How It Works

### 👨‍💼 Lenders

* Deposit crypto assets into the lending pool.
* Earn passive income through interest.

### 👨‍💻 Borrowers

* Provide cryptocurrency as collateral.
* Borrow assets against their collateral.

### 🤖 Smart Contracts

* Verify collateral.
* Issue loans automatically.
* Calculate interest.
* Handle repayments and liquidations.
